Swarm

  • Second complete project in Unity, first original game.

    In swarm, the player controls a tank in an alien world, where hundreds of bug-like enemy units are spawned and swarm the player!

    Swarm is a game stylistically modeled after overhead 3rd person action shooter 'em ups. Similar to games like "Gauntlet" and "FORCED", the game spawns hordes of units from a various "spawners". These spawners are synced up to musical components, providing the soundtrack for the game as your shred through the swarm.

    Dev Time: 1.5 weeks (Oct-12 - Oct-21, 2015)

    The enemy units are controlled with a standard BOIDs code. Credit for the code goes to Introduction to Game Design, Prototyping, and Development (Part II, Ch. 26), by Jeremy Gibson-Bond. By giving each different unit type a few different parameters, unique behaviours for each type of enemy unit were attained.
